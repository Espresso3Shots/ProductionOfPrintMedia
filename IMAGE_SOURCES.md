# Image Sources Guide

คู่มือนี้ใช้เก็บแหล่งภาพที่เหมาะกับบทเรียน `การผลิตสื่อสิ่งพิมพ์` โดยเน้น 3 เรื่อง:

1. หาแหล่งภาพให้ตรงกับหัวข้อของแต่ละหน่วย
2. ใช้ภาพที่มีสิทธิ์ใช้งานและเครดิตตรวจสอบได้
3. เก็บ metadata ให้พอสำหรับอ้างอิงในเว็บหรือเอกสารสอน

## หลักการคัดภาพ

- เริ่มจาก `CC0`, `Public Domain`, หรือแหล่งที่มี `item-level rights` ชัดก่อน
- เก็บอย่างน้อย: `ชื่อภาพ`, `ผู้สร้าง`, `สถาบัน`, `URL หน้ารายการ`, `license`, `วันที่ดาวน์โหลด`
- ถ้าใช้ภาพหน้าจอโปรแกรม ให้ถ่ายจากเครื่องที่ใช้สอนเองเป็นหลัก แล้วเก็บเวอร์ชันโปรแกรมกำกับไว้
- ถ้าใช้ภาพจาก `Wikimedia Commons` ให้เก็บทั้งหน้า `File:` และข้อมูล license ของไฟล์นั้น
- ถ้าใช้ภาพจาก `Unsplash` หรือ `Pexels` ให้เก็บหน้า license ของแพลตฟอร์มไว้คู่กับ URL ภาพต้นทางเสมอ

## แนวทางอัปเดตครั้งต่อไป

- ดาวน์โหลดภาพหน้าจอและภาพอ้างอิงจริงเพิ่มสำหรับหน่วยที่ยังต้องการตัวอย่างเฉพาะทาง แล้วเก็บ metadata เป็นไฟล์ JSON
- ผูกเครดิตภาพกับหน้าเว็บโดยอัตโนมัติจากฐานข้อมูลกลาง
- ขยายคลังภาพจริงที่มี provenance ชัดให้ครอบคลุมทั้งภาพเครื่องมือ ภาพชิ้นงาน และภาพกระบวนการผลิต
